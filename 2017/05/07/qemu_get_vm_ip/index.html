获取qemu创建的虚拟机的IP地址

作者： 耗子007

先用virsh net-list获取虚拟机的网络列表

然后获取IP的分配列表

获取虚拟机的列表

[root@localhost vms]# virsh list
Id Name State
—————————————————-
12 main running
13 slaver1 running
14 slaver2 running

注：virsh net-dhcp-leases 网络名：会把该网络上分配出去的IP地址显示出来

获取制定虚拟机的IP

virsh edit vmID可以看到虚拟机的MAC地址，然后到第二步中找到对应的IP。

如果有更好的办法可以联系我：duguhaotian@gmail.com
